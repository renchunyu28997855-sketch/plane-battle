# 飞机大战 (Plane Battle)

![TypeScript](https://img.shields.io/badge/TypeScript-5.3-blue)
![Vite](https://img.shields.io/badge/Vite-5.0-purple)
![License](https://img.shields.io/badge/License-MIT-green)

一款使用 HTML5 Canvas + TypeScript 开发的经典飞机大战游戏。

## 🎮 游戏特性

### 5 个关卡
- **第一关** - 50 秒，175 架敌机
- **第二关** - 50 秒，250 架敌机
- **第三关** - 50 秒，300 架敌机
- **第四关** - 60 秒，360 架敌机
- **第五关** - 80 秒，420 架敌机（Boss 关卡）

### 4 种子弹类型 × 5 级升级

| 子弹类型 | 等级 1 | 等级 2 | 等级 3 | 等级 4 | 等级 5 |
|---------|--------|--------|--------|--------|--------|
| **普通子弹** | 1x 尺寸 | 2x 尺寸 | 4x 尺寸 | 7x 尺寸 | 10x 尺寸 |
| **散弹** | 2 发 | 3 发 | 4 发 | 5 发 | 6 发 |
| **追踪子弹** | 速度+15%<br>尺寸+50% | 速度+30%<br>尺寸+100% | 速度+45%<br>尺寸+150% | 速度+60%<br>尺寸+200% | 速度+75%<br>尺寸+250% |
| **激光子弹** | 宽度+20%<br>速度+50% | 宽度+40%<br>速度+100% | 宽度+60%<br>速度+150% | 宽度+80%<br>速度+200% | 宽度+100%<br>速度+250% |

> 💡 切换子弹类型时会保留当前等级

### 7 种敌人类型

- **普通敌机** - 标准敌人
- **快速敌机** - 移动速度快，威胁高
- **重型敌机** - 血量高，难以击毁
- **精英敌机** - 高血量 + 高速度
- **护盾敌机** - 带有护盾，需多次攻击
- **轰炸机** - 发射子弹
- **Boss** - 第五关出现的大型敌人

### 6 种道具

| 道具 | 效果 |
|------|------|
| 🛡️ 护盾 | 增加 1 层护盾（最多 5 层） |
| 🔹 普通子弹 | 切换为普通子弹 + 升级 |
| 🔶 散弹 | 切换为散弹 + 升级 |
| 🔷 追踪子弹 | 切换为追踪子弹 + 升级 |
| 🔴 激光子弹 | 切换为激光子弹 + 升级 |
| 💰 金币 | 增加分数 |

### 操作方式

- **方向键** - 移动飞机
- **空格键** - 发射子弹
- **P 键** - 暂停/继续游戏

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览构建结果

```bash
npm run preview
```

## 📁 项目结构

```
src/
├── assets/          # 静态资源 (图片、音频)
├── components/      # UI 组件
├── entities/        # 游戏实体
│   ├── bullet.ts           # 子弹类
│   ├── bullet-manager.ts  # 子弹管理器
│   ├── enemy.ts           # 敌机类
│   ├── enemy-manager.ts   # 敌机管理器
│   ├── explosion.ts       # 爆炸效果
│   ├── player.ts          # 玩家飞机
│   ├── prop.ts            # 道具
│   └── prop-manager.ts    # 道具管理器
├── engine/          # 游戏引擎核心
│   ├── game-engine.ts     # 主游戏引擎
│   └── renderer.ts        # 渲染器
├── types/           # 类型定义
│   └── index.ts
└── main.ts          # 入口文件
```
